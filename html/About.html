The Automa Project

Automa is the brainchild of Bayrem Hanini, developed entirely as a solo project throughout 2025. What began as a simple idea to automate repetitive tasks like "bulk mail sending" as a starter then it became a bigger project to automate other tasks than mails.

Development Journey

Built from the ground up, every aspect of Automa - from the initial design to the final implementation - was personally crafted by Bayrem hanini. The project represents over 5 months of dedicated development, testing, and refinement.

Technical Implementation

Automa was built using a modern web technology stack:

HTML5 CSS JavaScript Node.js Express MongoDB spline Python

Project Philosophy

"Automa was born from the frustration with repetitive digital tasks. I wanted to create tools that would handle the boring work so people could focus on what matters. Every line of code reflects my belief that technology should serve people, not the other way around."
